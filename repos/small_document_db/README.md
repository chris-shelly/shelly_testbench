# SmallDoc DB
Lightweight, pure Python, document oriented database for Python applications to store small amounts of data.

Similar to MongoDB, data is stored in "documents" as python `dict`s. 

Allows for:
- CRUD
  - Insert
  - Update
  - Upsert
  - Read
  - Delete
- Storing Data collections as Tables
- Queries
  - search for field values
  - combine queries with logic
- Transformations/Data edits
  - use a 'map' function
